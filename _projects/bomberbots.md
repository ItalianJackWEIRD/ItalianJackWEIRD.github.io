## Overview

This project is a Bomberman clone made in Lua using the Löve2D framework. My classmate and I worked on it while studying game development.

## What I Built

- Grid-based movement and collision system
- Bomb placement, explosion propagation logic
- AI opponents with pathfinding
- Multiplayer game loop and win conditions
- Tilemap rendering and level design tools

## What I Learned

Collaborating on a shared codebase with version control, designing clean interfaces between systems, and balancing game feel with technical constraints.
