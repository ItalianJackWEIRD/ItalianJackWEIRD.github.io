## Overview

This project is a take on the Space Shooter genre. I developed my own ECS (Entity-Component System) and used the Wave Function Collapse algorithm as a procedural background generator.

## What I Built

- Custom ECS architecture for managing game entities efficiently
- Wave Function Collapse implementation for generating varied, coherent backgrounds
- Collision detection and response system
- Enemy AI with different behavior patterns
- Score tracking and game state management

## What I Learned

Building this project taught me a lot about data-oriented design and cache-friendly programming. The Wave Function Collapse algorithm was a fascinating challenge — getting it to generate levels that felt both random and coherent required a lot of tweaking.
